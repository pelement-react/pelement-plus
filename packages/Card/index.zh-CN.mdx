将信息聚合在卡片容器中展示。

## 基础用法

卡片包含标题，内容以及操作区域。

Card 组件由 header body 和 footer组成。 header 和 footer是可选的

<CodeBlock name="CardBasic">
  <CardBasic />
</CodeBlock>

## 简单卡片

卡片可以只有内容区域。

<CodeBlock name="CardSimple">
  <CardSimple />
</CodeBlock>

## 有图片内容的卡片

可配置定义更丰富的内容展示。

配置 body-style 属性来自定义 body 部分的样式。 在这个例子中我们还使用了 Col 组件来布局。

<CodeBlock name="CardBodyStyle">
  <CardBodyStyle />
</CodeBlock>

## 带有阴影效果的卡片

你可以定义什么时候展示卡片的阴影效果。

通过 shadow 属性设置卡片阴影出现的时机。 该属性的值可以是：always、hover 或 never。

<CodeBlock name="CardShadow">
  <CardShadow />
</CodeBlock>

## API

### Attributes

| 属性名    | 说明              | 类型                    | 默认值 |
| --------- | ----------------- | ----------------------- | ------ |
| header    | 卡片的标题        | string &#124; ReactNode | —      |
| footer    | 卡片页脚          | string &#124; ReactNode | —      |
| bodyStyle | body 的 CSS 样式  | CSSProperties           | —      |
| bodyClass | body 的自定义类名 | style &#124; style[]    | —      |
| shadow    | 卡片阴影显示时机  | CSSProperties           | —      |
