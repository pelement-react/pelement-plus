按钮和图标上的数字或状态标记。

## 基础用法

可以用来展示新消息的数量。

数量值可接受 Number 或 String。

<CodeBlock name="BadgeBasic">
  <BadgeBasic />
</CodeBlock>

## 最大值

你还可以自定义最大值

由 max 属性定义，接受 Number 值。 请注意，仅在值也是 Number 时起作用。

<CodeBlock name="BadgeMax">
  <BadgeMax />
</CodeBlock>

## 自定义显示内容

你也可以展示除数字以外你想要展示的任何值。

当 value 是 String 时，可以显示自定义文字。

<CodeBlock name="BadgeValue">
  <BadgeValue />
</CodeBlock>

## 小红点

通过一个小红点标记来告知用户有新内容。

使用 is-dot 属性。 是个布尔值。

<CodeBlock name="BadgeIsDot">
  <BadgeIsDot />
</CodeBlock>

## API

### Attributes

| 属性名 | 说明                                                              | 类型                                                                      | 默认值 |
| ------ | ----------------------------------------------------------------- | ------------------------------------------------------------------------- | ------ |
| value  | 显示值                                                            | string / number                                                           | ''     |
| max    | 最大值，超过最大值会显示 max+。 只有当 value 是数字类型时起作用。 | number                                                                    | 99     |
| isDot  | 是否显示小圆点。                                                  | boolean                                                                   | false  |
| type   | badge type.                                                       | 'primary' &#124; 'success' &#124; 'warning' &#124; 'danger' &#124; 'info' | danger |
