## Statistic API

### Statistic Attributes

| 属性名           | 说明             | 类型                                    | 默认值 |
| ---------------- | ---------------- | --------------------------------------- | ------ |
| value            | 数字内容         | number                                  | 0      |
| decimalSeparator | 设置小数点符号   | string                                  | .      |
| formatter        | 自定义数字格式化 | (value: number) => string &#124; number | —      |
| groupSeparator   | 设置千分位标识符 | string                                  | ,      |
| precision        | 数字精度         | number                                  | 0      |
| prefix           | 设置数字的前缀   | string                                  | —      |
| suffix           | 设置数字的后缀   | string                                  | —      |
| title            | 数字标题         | string &#124; ReactNode                 | —      |
| valueStyle       | 数字样式         | CSSProperties                           | —      |

## Countdown API

### Countdown Attributes

| 属性名     | 说明             | 类型                    | 默认值   |
| ---------- | ---------------- | ----------------------- | -------- |
| value      | 目标时间         | number / Dayjs          | —        |
| format     | 格式化倒计时     | string                  | HH:mm:ss |
| prefix     | 设置倒计时前缀   | string                  | —        |
| suffix     | 设置倒计时的后缀 | string                  | —        |
| title      | 倒计时标题       | string &#124; ReactNode | —        |
| valueStyle | 倒计时值的样式   | CSSProperties           | —        |

### Countdown Events

| 名称     | 描述           | 类型                    |
| -------- | -------------- | ----------------------- |
| onChange | 时间差改变事件 | (value: number) => void |
| onFinish | 倒计时结束事件 | () => void              |
