## 基础用法

由 `type` 属性来选择 tag 的类型。 也可以通过 `color` 属性来自定义背景色。

<CodeBlock name="TagBasic">
  <TagBasic />
</CodeBlock>

## 可移除标签

设置 `closable` 属性可以定义一个标签是否可移除。 它接受一个 `Boolean`。 默认的标签移除时会附带渐变动画。 如果不想使用，可以设置 `disable-transitions` 属性，它接受一个 `Boolean`，`true` 为关闭。 当 Tag 被移除时会触发 `close` 事件。

<CodeBlock name="TagClosable">
  <TagClosable />
</CodeBlock>

## 不同尺寸

Tag 组件提供除了默认值以外的三种尺寸，可以在不同场景下选择合适的按钮尺寸。

使用 `size` 属性来设置额外尺寸, 可选值包括 `large`, `default` 或 `small`.

<CodeBlock name="TagSize">
  <TagSize />
</CodeBlock>

## 主题

Tag 组件提供了三个不同的主题：`dark`、`light` 和 `plain`。

通过设置 effect 属性来改变主题，默认为 `light`。

<CodeBlock name="TagEffect">
  <TagEffect />
</CodeBlock>

## 圆形标签

Tag 可以向按钮组件一样变为完全圆形。

<CodeBlock name="TagRound">
  <TagRound />
</CodeBlock>

## API

### Attributes

| 属性名   | 说明           | 类型                                                               | 默认值 |
| -------- | -------------- | ------------------------------------------------------------------ | ------ |
| type     | Tag 的类型     | 'success' &#124; 'info' &#124; 'warning' &#124; 'danger' &#124; '' | ''     |
| closable | 是否可关闭     | `boolean`                                                          | false  |
| size     | Tag 的尺寸     | 'large' &#124; 'default' &#124; 'small' &#124; ''                  | ''     |
| effect   | Tag 的主题     | 'dark' &#124; 'light' &#124; 'plain'                               | light  |
| round    | Tag 是否为圆形 | `boolean`                                                          | false  |
