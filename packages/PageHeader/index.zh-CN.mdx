有时，页面全是元素，您可能不想展示页面上方的图标，您可以设置icon属性值为null来去除它。

## API

### Attributes

| 属性名     | 说明                              | 类型                    | 默认值 |
| ---------- | --------------------------------- | ----------------------- | ------ |
| icon       | Page Header 的图标 Icon 组件      | ReactNode               | Back   |
| title      | Page Header 的主标题，默认是 Back | string &#124; ReactNode | Back   |
| content    | Page Header 的内容                | string &#124; ReactNode | —      |
| extra      | Page Header 的扩展设置            | string &#124; ReactNode | —      |
| breadcrumb | Page Header 的面包屑导航内容      | string &#124; ReactNode | —      |

### 事件

| 事件名 | 说明             | 类型       |
| ------ | ---------------- | ---------- |
| onBack | 点击左侧区域触发 | () => void |
