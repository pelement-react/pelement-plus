## Link API

### Attributes

| 属性名    | 说明           | 类型                                                         | 默认值  |
| --------- | -------------- | ------------------------------------------------------------ | ------- |
| type      | 类型           | 'primary'  'success'  'warning'  'danger'  'info'  'default' | default |
| underline | 是否下划线     | `boolean`                                                    | true    |
| disabled  | 是否禁用状态   | `boolean`                                                    | false   |
| href      | 原生 href 属性 | `string`                                                     | —       |
| icon      | icon           | `string` / `Component`                                       | —       |
