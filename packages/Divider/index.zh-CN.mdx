## API

### Attributes

| 属性名           | 说明                   | 类型                                                           | 默认值     |
| ---------------- | ---------------------- | -------------------------------------------------------------- | ---------- |
| direction        | 设置分割线方向         | 'horizontal' / 'vertical'                                      | horizontal |
| border-style     | 设置分隔符样式         | 'none' / 'solid' / 'hidden' / 'dashed' / ...(css/border-style) | solid      |
| content-position | 自定义分隔线内容的位置 | 'left' / 'right' / 'center'                                    | center     |
