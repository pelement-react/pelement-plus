文本的常见操作

## 基础用法

由 `type` 属性来选择 Text 的类型。

<CodeBlock name="TextBasic">
  <TextBasic />
</CodeBlock>

## 尺寸

使用 `size` 属性配置尺寸，可选的尺寸大小有: `large`, `default` 或 `small`

<CodeBlock name="TextSize">
  <TextSize />
</CodeBlock>

## 省略

通过 `truncated` 属性，在文本超过视图或最大宽度设置时展示省略符。 通过 `line-clamp` 属性控制多行的样式

<CodeBlock name="TextTruncated">
  <TextTruncated />
</CodeBlock>

## API

### Attributes

| 属性名    | 说明       | 类型                                                  | 默认值  |
| --------- | ---------- | ----------------------------------------------------- | ------- |
| type      | 类型       | 'primary' / 'success' / 'warning' / 'danger' / 'info' | —       |
| size      | 大小       | 'large' / 'default' / 'small'                         | default |
| truncated | 显示省略号 | `boolean`                                             | false   |
| lineClamp | 最大行数   | `number`                                              | —       |
